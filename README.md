# GOTH Starter Project

A starter template project using Golang, Alpine.js, Templ, Tailwind CSS, Pines UI, and HTMX. Created while following AnthonyGG's YouTube video  [Ultimate Golang-HTMX-TEMPL-TAILWIND Project Template](https://www.youtube.com/watch?v=1dqp1s72Z8E)

## Golang

1. [Golang](https://go.dev/)

## External Packages

1. [Alpine.js](https://alpinejs.dev/)
2. [Templ](https://github.com/a-h/templ)
3. [Tailwind](https://tailwindcss.com/)
4. [Pines UI](https://devdojo.com/pines)
5. [HTMX](https://htmx.org)
6. [Air](https://github.com/cosmtrek/air)
